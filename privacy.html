1. Ruang Lingkup

Kebijakan ini berlaku untuk:

  • Aplikasi Flutter Nimexx (Android/iOS)
  • Situs web Nimexx dan subdomain terkait

2. Data yang Kami Kumpulkan

Untuk menyediakan layanan terbaik, kami mengumpulkan:

  • Data Akun: Email, nama pengguna, dan foto profil (opsional)

3. Keamanan Data

Kami menggunakan Firebase dengan proteksi:

  • Enkripsi data transit (SSL/TLS)
  • Aturan keamanan Firebase yang ketat
  • Autentikasi berbasis OAuth 2.0

4. Panel Admin

Fitur admin kami mengelola:
